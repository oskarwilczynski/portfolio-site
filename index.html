PROJECTS

POMODORO TIMER

Pomodoro Timer was my first project ever. I built it with pure HTML, CSS and JavaScript - no frameworks, no libraries. It was very important to me to get confident with basics, and fully understand them, before moving further, to jQuery and Bootstrap for example.

The timer is capable of saving desired time on each mode before starting the clock and has a pause function. Also, the app is fully responsive and functional on all devices.

FITNESS DATABASE

Fitness Database is quite simple "search engine" that lets a search any exercise listed in open-source API. The site displays description and, if available, image of the requested exercise in one of the supported language. It also features autocompletion, so the user doesn't need to know exact name of the exercise.

With this project I learnt a lot - jQuery, AJAX, JSON and usage of external APIs.
